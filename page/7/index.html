Устранение проблем с русскими символами в git
10 ноября 2009



Когда вы начнете работать с версией git под windows в командной строке, вы столкнётесь со следующей проблемой -- все сообщения git, в которых фигурируют русские символы будут нечитаемы. Имена файлов, на русском языке, будут выглядеть так -- "\362\345\361\362", а тексты коммитов примерно так -- <C8><ED><E8><F6><E8><E0><EB><E8><E7><E0><F6><E8><FF> <EF><F0><EE><E5><EA><F2><E0>.
